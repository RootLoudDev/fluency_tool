Save a Recording on Your Own Hard Drive

  • Record a message by clicking the red circle button.
  • Then click here to call saveRecordedMessage() from JavaScript.
  • A file save dialog will appear.
  • You can save the file anywhere you wish on your hard drive.

The JavaScript code looks like this: 

 document.ListenUpRecorder.saveRecordedMessage();
