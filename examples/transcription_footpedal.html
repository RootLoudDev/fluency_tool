Enable Transcription Features

Note: Footpedal Requires Windows OS and Sun Java!

Use this Applet parameter to add foot pedal support: 
<param name="useFootPedal" value="yes">

Also use the "ListenUpTranscriber.jar" in the ARCHIVES list in the Applet tag instead of "JavaSonicsListenUp.jar".

Please note that use of these transcription features requires a special license. Please contact us for details.

Foot Pedals

The first time you run this application you will be asked to install a FootPedalServer native library. This will only work on Windows OS. If you do not have the foot pedal then just skip the installation of the library. We support the Infinity USB Footpedal available from Aaaprice, Sylvan and Xkeys.

Note that when you lift your foot the cursor will rewind a few seconds because we are using the "autoBackStep" feature.
