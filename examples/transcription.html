Enable Transcription Features

You can enable additional features designed to make transcription easier. 

<param name="transcription" value="yes">
<param name="showWaveform" value="yes">

Also use the "ListenUpTranscriber.jar" in the ARCHIVES list in the Applet tag instead of "JavaSonicsListenUp.jar".

Please note that use of these transcription features requires a special license. Please contact us for details.

The transcription option adds extra buttons for transport control. On later pages you will also see how to use the foot pedal, Philips Speech Mike, and Hot Keys.

See more examples.

Using the ListenUp Transcription Controls

Use the Record button to record a long audio message. Then use the Transcriber controls to play it with various options.

Play Button

While playing, you may vary the speed of the playback using the graphic scrollbar, slowing the playback down to 70% its normal speed, or speeding it up to 120% its normal speed.  Moving the scrollbar while fast forwarding or rewinding will not affect fast forward or rewind speed; it sets playback speed only.

Pause Button

The Pause button will stop playback at the current position. You can request that it automatically rewind a few seconds by using the "autoBackStep" parameter. Here we have set "autoBackstep" to 3.0 seconds.

Fast Forward

Activate either by clicking the graphic button with the mouse. Clicking the graphic fast forward button during fast forward play advances to the next fast forward speed.

The speeds are:

  1. 150% faster than original speed
  2. 200% faster than original speed
  3. 300% faster than original speed
  4. Skip mode: skips forward rapidly, playing little bits of audio at normal speed. This emulates fast forward in CD and DVD players. Clicking the graphic button again while in skip mode drops the speed down to 150% again, cycling from the beginning again. The fast forward button remembers that last speed it was set at.

Rewind

Activate either by clicking the graphic button with the mouse. Clicking the graphic rewind button during rewind playback advances to the next rewind speed.

The speeds are:

  1. 150% faster than original speed
  2. 200% faster than original speed
  3. 300% faster than original speed
  4. Skip mode: skips backwards rapidly, playing little bits of audio at normal speed. This emulates rewind in CD and DVD players. Clicking the graphic button again while in skip mode drops the speed down to 150% again, cycling from the beginning again. The rewind button remembers that last speed it was set at.
