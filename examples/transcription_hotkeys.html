Transcription with Hot Keys

Note: HotKey Feature Requires Windows OS and Sun Java!

Use this Applet parameter to add "Hot Keys" support: 
<param name="playHotKey" value="p+alt+shift">
<param name="recordHotKey" value="r+alt+shift">
<param name="pauseHotKey" value="a+alt+shift">
<param name="stopHotKey" value="s+alt+shift">
<param name="rewindHotKey" value="w+alt+shift">
<param name="forwardHotKey" value="f+alt+shift">
<param name="toEndHotKey" value="e+alt+shift">
<param name="toBeginHotKey" value="b+alt+shift">

Also use the "ListenUpTranscriber.jar" in the ARCHIVES list in the Applet tag instead of "JavaSonicsListenUp.jar".

Please note that use of these transcription features requires a special license. Please contact us for details.

You Decide which Hot Keys can Control ListenUp

Hot Keys can be set to any combination using Applet parameters. For this test hold down ALT and SHIFT modifier keys then press the letter key.

  • ALT+SHIFT + R = Record
  • ALT+SHIFT + P = Play
  • ALT+SHIFT + A = pAuse
  • ALT+SHIFT + S = Stop
  • ALT+SHIFT + W = reWind
  • ALT+SHIFT + F = Fast Forward
  • ALT+SHIFT + E = to End
  • ALT+SHIFT + B = to Begin

Try this:

  1. First press ALT+SHIFT+R to start recording.
  2. Press ALT+SHIFT+S to stop.
  3. Press ALT+SHIFT+P to play back.
  4. Press ALT+SHIFT+W to rewind.
  5. Try the other hot keys..
