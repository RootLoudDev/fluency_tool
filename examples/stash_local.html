Save files locally using "stashBeforeSend"

Sometimes a user could lose a recording if the server or the network was down and they could not upload the recording. You can request that a copy of the recording be stashed on the local hard drive by setting the "stashBeforeSend" parameter. 

<param name="stashBeforeSend" value="yes" />

We say "stash" instead of "save" because the file is not permanently saved. It will be kept for several days then erased. This is to prevent the users drive from filling up with old files. Try this:

  1. Right click on waveform. (On a Mac with a one-button mouse use "Control+click".)
  2. The very first time you do this you will probably see the "Load Most Recently Stashed Recording" item is disabled.
  3. Record a message then hit "Send".
  4. You will see an error message saying the recording was not uploaded. This is a simulated error.
  5. Click here to reload this page. Notice the recording is gone.
  6. Right click on waveform.
  7. Select the "Load Most Recent Recording" item.
  8. Your previous recording will be loaded.
