These simple examples show how to use specific features of ListenUp.

Basic Play and Record Features

  • Play a voice message: play_welcome.html
  • Record audio and upload a WAV file: record_upload_wav.html
  • Use a waveform display to show recorded audio: gui_waveform.html
  • Waveform editing: record_editor.html
  • Modify an existing recording: record_insert.html
  • Enable transcription features: transcription.html

Advanced Features

  • Add text fields so user can enter additional information: use_text_fields.html
  • Use JavaScriptand an HTML FORM with ListenUp: send_using_javascript.html
  • Trigger JavaScript from server using CALL JS: record_upload_calljs.html
  • Changed to wide or tall arrangement: gui_arrangements.html
  • Change width or remove bevel from Applet: gui_bevel.html
  • Put ListenUp in its own window: gui_dialog.html
  • Change the buttons or colors: gui_skin.html
