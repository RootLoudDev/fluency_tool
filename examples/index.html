These simple examples show how to use specific features of ListenUp. View the HTML source code to see how it is done.

Basic Play and Record Features

  • Play a voice message - play_welcome.html
  • Record audio and upload a WAV file - record_upload_wav.html
  • Waveform editing - record_editor.html
  • Modify an existing recording - record_insert.html
  • Stash a recording on the local hard drive - stash_local.html

JavaScript Related Features

  • Use JavaScript and an HTML FORM with ListenUp - send_using_javascript.html
  • Trigger JavaScript from server using CALLJS - record_upload_calljs.html
  • Workaround for "Click to activate" message in IE6 and IE7 - record_auto_activate.html
  • Save a recording to local hard drive - save_recorded_message.html
  • Request state change events from the ListenUp Applet - request_state_changes.html

GUI Features

  • Add text fields so user can enter additional information - use_text_fields.html
  • Changed to wide or tall arrangement - gui_arrangements.html
  • Change width or remove bevel from Applet - gui_bevel.html
  • Put ListenUp in its own window - gui_dialog.html
  • Change the buttons or colors - gui_skin.html

Transcription Features

These features require the "transcription" license option. There is an additional fee for this option. The free demo license includes the transcription option.

  • Basic transcription GUI - transcription.html
  • Playing back with foot pedal - transcription_footpedal.html
  • Recording and playing with hot keys - transcription_hotkeys.html
