Using JavaScript to Send a Recording with a FORM

This demonstrates the combination of an HTML FORM and the ListenUp Applet. JavaScript calls the ListenUp Applet method to upload the recording and wait for a result. It then sends the FORM.

This HTML FORM uses JavaScript to call the Java Applet. It requires LiveConnect, which may only work on Windows, or on Mac with Firefox or Safari 2.0.2 or later!.
